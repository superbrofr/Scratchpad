1. Run the AuthenticationServer
2. Run the FileServer
3. Run the Client
4. Use the <connect> command to enter the IP address or hostname of the FileServer
5. Use the <login> command to login the the FileServer with a username and password
	Username: user
	Password: pass
6. Use the <list> command to list the available files on the FileServer
7. Use the <get> command to specify the name of the file you want to download,
	then enter the name that you wish to save the file as, including file extension
8. Use the <disconnect> command to disconnect from the FileServer
