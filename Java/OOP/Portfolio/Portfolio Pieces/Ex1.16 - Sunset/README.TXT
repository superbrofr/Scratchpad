Project "picture"
Authors: Michael Kolling and David J. Barnes with extension by Charlotte Pierce

The extension in this project is part of the material for the Object-Oriented
programming portfolio of Charlotte Pierce.
    
This project demonstrates the ability to read, understand, and modify other's code
in order to perform a new function. In this application, the new function is 
causing the sun object to move down the screen in the manner of a sunset.

To see this extension:
- create an instance of Picture
- execute the draw() method
- execute the sunset() method
